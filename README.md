# MSCM-Net for pCR Prediction

The code for training, testing, and visualization of the MSCM-Net model for pathological complete response (pCR) prediction from 3D CT images.

---

## Requirements

monai  
numpy  
torch  
ignite  
scipy  
matplotlib  
pytorch-grad-cam  


## Getting Started

### Installation

Create and activate a new conda environment

---

## Training

Run the training script: 

```python train.py```
## Testing

Run the testing script: 

```python test.py```
## Grad-CAM Visualization

To generate Grad-CAM3D visualization: 

```python GradCAM3D.py```
